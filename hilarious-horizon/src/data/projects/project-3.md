This document teaches you **common Korean phrases** for daily conversation.

- **죄송합니다 (Joesonghamnida)**: I'm sorry (formal)
- **미안해요 (Mianhaeyo)**: Sorry (polite)
- **괜찮아요 (Gwaenchanhayo)**: It's okay / No problem
- **이해했어요? (Ihaehaesseoyo?)**: Did you understand?
- **잘 지내세요? (Jal jinaeseyo?)**: How are you? (formal)

> Tip: Practice using these phrases in roleplay conversations with a friend.
